## Project setup
```
composer install

```
copy the .env.example file to a .env file, and set the database config


## execute
```
npm install

```
and

```
 npm run dev
```


## to create tables and seed example data
```
php artisan migrate:fresh --seed
```

### to run the app
```
php artisan serve
```
